# i-learn-rust

A learning repository for mastering Rust through building small projects and tools.

## Overview

This repository contains various CLI tools and utilities built with Rust, designed as hands-on learning exercises to deepen knowledge of the Rust programming language.

## Projects

### 1. LogPeek

A small Rust CLI tool for quickly inspecting logs, filtering results, and extracting useful summaries.

**Features:**
- Fast log inspection
- Advanced filtering capabilities
- Summary extraction from log data

## Getting Started

Each project has its own directory with its own `Cargo.toml` configuration.

### Building

```bash
cargo build
```

### Running

```bash
cargo run
```

## Goal

The primary goal of this repository is to learn and practice Rust by building practical tools that solve real-world problems in software development.
